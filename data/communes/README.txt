This directory is completely independent of the application logic.

This directory can be used to:

    Import 1541 communes from "algeria_cities.json" to the database by running:
        node import-communes.js --import (or --delete to delete all the communes)
    Import 58 wilayas from "wilayas.json" to the database by running:
        node import-wilayas.js --import (or --delete to delete all the wilayas)
